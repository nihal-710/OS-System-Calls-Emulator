# OS System Calls Emulator

An interactive web-based emulator that demonstrates how **system calls act as the interface between user programs and the operating system kernel**.

The project simulates common system calls and visualizes how processes interact with system resources such as files and memory.

## Live Demo
https://nihal-710.github.io/OS-System-Calls-Emulator/

## Features
- Simulation of common **system calls** (`open`, `read`, `write`, `fork`, `exec`, `exit`, `malloc`)
- Visualization of **user space vs kernel space**
- Dynamic **process table representation**
- Terminal-style interface for executing simulated system calls

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project was built to help understand core **Operating System concepts** such as system calls, process management, and resource interaction between user programs and the OS kernel.
